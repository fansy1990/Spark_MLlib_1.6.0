Welcome to Spark Alogrithm Show Platform


    Spark Alogirhtm 算法集锦，包括算法（+表示已经实现，-表示未实现）：
  • 分类与回归（Classification and regression）
    • vectordump : Dump vectors from a sequence file to text -
    • clusterdump : Dump cluster output to text +
    • seqdumper : Generic Sequence File dumper +
    • lucene.vector : Generate Vectors from a Lucene index -
    • arff.vector : Generate Vectors from an ARFF file or directory +
    • rowid : Map SequenceFile<Text,VectorWritable> to {SequenceFile<IntWritable,VectorWritable>, SequenceFile<IntWritable,Text>} -
    • split : Split Input data into test and train sets -
    • matrixdump : Dump matrix in CSV format -
    • regexconverter : Convert text files on a per line basis based on regular expressions -
    • seqdirectory : Generate sequence files (of Text) from a directory -
    • seq2sparse: Sparse Vector generation from Text sequence files -
    • seq2encoded: Encoded Sparse Vector generation from Text sequence files -
    • seqwiki : Wikipedia xml dump to sequence file -
    • seqmailarchives : Creates SequenceFile from a directoy containing gzipped mail archives -
    • lucene2seq : Generate Text SequenceFiles from a Lucene index -
    • concatmatrices : Concatenates 2 matrices of same cardinality into a single matrix -
    • resplit : Splits a set of SequenceFiles into a number of equal splits +
    • qualcluster : Runs clustering experiments and summarizes results in a CSV -
    • describe : Describe the fields and target variable in a data set +
  • Math
    • transpose : Take the transpose of a matrix -
    • matrixmult : Take the product of two matrices -
    • svd : Lanczos Singular Value Decomposition -
    • cleansvd : Cleanup and verification of SVD output -
    • rowsimilarity : Compute the pairwise similarities of the rows of a matrix -
    • vecdist : Compute the distances between a set of Vectors (or Cluster or Canopy, they must fit in memory) and a list of Vectors +
    • ssvd : Stochastic SVD -
  • Clustering
    • kmeans : K-means clustering +
    • fkmeans : Fuzzy K-means clustering +
    • cvb : LDA via Collapsed Variation Bayes (0th deriv. approx) -
    • ...
  • Classification
    • trainnb : Train the Vector-based Bayes classifier +
    • testnb : Test the Vector-based Bayes classifier +
    • buildforest : Build the random forest classifier +
    • testforest : Test the random forest classifier +
  • SGD -
    • trainlogistic : Train a logistic regression using stochastic gradient descent
    • runlogistic : Run a logistic regression model against CSV data
    • cat : Print a file or resource as the logistic regression models would see it
    • trainAdaptiveLogistic : Train an AdaptivelogisticRegression model
    • validateAdaptiveLogistic : Validate an AdaptivelogisticRegression model against hold-out data set
    • runAdaptiveLogistic : Score new production data using a probably trained and validated AdaptivelogisticRegression model
  • HMM -
    • baumwelch : Baum-Welch algorithm for unsupervised HMM training
    • viterbi : Viterbi decoding of hidden states from given output states sequence
    • hmmpredict : Generate random sequence of observations by given HMM
  • Classifier Utils -
    • cmdump : Dump confusion matrix in HTML or text formats
  • Recommenders
    • splitDataset : split a rating dataset into training and probe parts +
    • evaluateFactorization : compute RMSE and MAE of a rating matrix factorization against probes +
    • itemsimilarity : Compute the item-item-similarities for item-based collaborative filtering +
    • recommenditembased : Compute recommendations using item-based collaborative filtering +
    • parallelALS : ALS-WR factorization of a rating matrix +
    • recommendfactorized : Compute recommendations using the factorization of a rating matrix +
